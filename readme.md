## Introduction

`coffeelib.templates` is a package with built-in templates for [coffeelib][coffeelib].
It gets pulled in when [coffeelib][coffeelib] gets installed with

```
npm install coffeelib -g
```

[coffeelib]: https://github.com/ch1c0t/coffeelib

## Development

`npm start` to start a development session.

`npm test` to run the tests of all templates.

`./test.coffee TEMPLATE_NAME` to run only the tests of one template. For example:

```
./test.coffee rpc
```
